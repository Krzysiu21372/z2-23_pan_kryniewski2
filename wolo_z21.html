Zastosowanie ID w CSS

Pojęcie Znaczenie pojęcia (uzupełnia uczeń)
Co znaczy skrót ID? Identyfikator (ang. Identifier)
Jaki jest cel stosowania ID? Unikalne oznaczenie elementu HTML, aby móc go precyzyjnie stylować lub odwoływać się do niego w JavaScript.
Jakie jest znaczenie znaku #? Znak # poprzedza nazwę ID w selektorze CSS, wskazując, że odnosimy się do elementu z tym konkretnym ID.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? W CSS: #nazwa_id { właściwości stylu }
W JavaScript: document.getElementById("nazwa_id")
Uwagi na temat stosowania ID (trzy konieczne wyliczenia)
  1. ID powinno być unikalne w obrębie całego dokumentu HTML.
  2. Nazwy ID powinny być opisowe, ale zwięzłe.
  3. Należy unikać nadmiernego stosowania ID, preferując klasy dla elementów, które mają wspólne style.
